Logiiels nécéssaires au projet :

- [Node.js](https://nodejs.org/)

## Installation

1. Clonez le dépôt :
   
   ```bash
   git clone https://github.com/username/nom-du-projet.git
2. Placez vous dans le bon répertoire :
   
   ```bash
   cd nomDuProjet
3. Installez les dépendances :
   
   ```bash
   npm i
4. Lancez le projet :
   
   ```bash
   npm run dev
